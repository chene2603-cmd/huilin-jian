# 《玄隐义客纪》架构总规范
# 版本: 1.0.0
# 生效日期: 2026-05-09

## 一、总体架构原则
1. 七层架构，严格单向依赖
2. 模块高内聚、低耦合
3. 国风主题优先，性能为基

## 二、技术栈规范
- 开发语言: C99 + 内联汇编(x64 SSE/AVX优化)
- 图形API: OpenGL 4.6 + 自定义国风渲染管线
- 构建系统: GNU Make + Windows Batch
- 目标平台: Windows 10+ (x64)

## 三、模块依赖规范
架构蓝图 → 核心代码 → 运维脚本 → 日志系统 → 补丁机制 → 交互界面 → 文档手册

## 四、数据流向规范
1. 渲染管线: CPU逻辑 → 世界数据 → 渲染队列 → GPU
2. 任务系统: 触发器 → 条件检查 → 状态更新 → 奖励发放
3. NPC AI: 感知输入 → 行为树评估 → 动作执行

## 五、性能指标
- 目标帧率: 60 FPS (1080P)
- 内存预算: 主内存 ≤ 8GB, VRAM ≤ 4GB
- 加载时间: 首次 ≤ 30s, 场景切换 ≤ 5s
