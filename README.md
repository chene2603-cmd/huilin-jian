# 《玄隐义客纪》 - 国风开放世界游戏

## 项目简介
《玄隐义客纪》是一款基于七层架构的国风开放世界游戏，以北宋汴京为背景，融合武侠、历史、奇幻元素。

## 技术架构
采用自研C/汇编游戏内核，七层解耦架构：

1. **架构蓝图层** - 工程规约、目录结构、数据流
2. **核心代码层** - C/汇编游戏内核、世界结构体
3. **运维脚本层** - 一键编译、资源打包、环境检测
4. **日志系统层** - 分级日志、错误分析、BUG清单
5. **补丁机制层** - 热更新、补丁管理、无需重编
6. **交互界面层** - CLI控制台、进度条、运维交互
7. **文档手册层** - 完整文档、教程、错误码表

## 快速开始

### 环境要求
- Windows 10+ 64位
- Visual Studio 2019+ (MSVC编译器)
- 8GB以上内存
- OpenGL 4.6+ 兼容显卡

### 一键编译
进入运维脚本目录

cd 3_运维脚本/build_scripts

检查环境

01_check_env.bat

编译发布版本

03_compile_release.bat

运行游戏

cd ../../build/release/bin

xuan_yin_yi_ke.exe
### 开发构建
编译调试版本

04_compile_debug.bat

运行测试

05_run_tests.bat

## 核心特性

### 🎮 游戏特色
- **开放世界汴京**: 真实还原北宋都城
- **国风渲染**: 丝绸、瓷器、水墨材质
- **动态NPC**: 自主AI、日程系统
- **分支任务**: 多结局、选择影响世界
- **武侠系统**: 内功、轻功、招式组合

### 🛠️ 工程优势
- **七层架构**: 模块解耦，独立演进
- **热更新**: 无需重编，在线修复
- **自检系统**: 启动自动验证数据
- **内存安全**: 定制内存池，泄露检测
- **优化内核**: SSE/AVX加速，汇编优化

## 开发指南

### 新增NPC
1. 在 `2_核心代码/src/world/chapter_bianjing/npc_data.c` 添加NPC数据
2. 在 `2_核心代码/src/world/chapter_bianjing/chapter_data.c` 注册
3. 运行ID冲突检查确保无冲突
4. 编译测试

### 新增任务
1. 在 `2_核心代码/src/world/chapter_bianjing/quest_data.c` 添加任务
2. 配置关联NPC和物品
3. 运行数据验证
4. 编译测试

### 制作补丁
生成NPC台词补丁

python 5_补丁机制/patch_generator/npc_dialog_gen.py
